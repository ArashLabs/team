Active Directory Replication

Tutorial – Active Directory Health Check

Objective

This check verifies that Active Directory replication is working correctly between all Domain Controllers.

Prerequisites

  • Access to a Domain Controller
  • Administrator permissions

Run Replication Check

Open Command Prompt and run:

CMD 
repadmin /replsummary

Verify that:

  • No replication failures are displayed
  • Failure percentage is 0%
  • All Domain Controllers are reachable

Result Interpretation

  • ✅ OK – Replication is healthy.
  • ⚠️ WARNING – Minor replication issues detected.
  • ❌ NOK – Replication failures detected.
